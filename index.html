Usage Example

You can load the icon Lua scripts directly in Roblox using loadstring and game:HttpGet. Replace iconname with the icon you want: 

loadstring(game:HttpGet("https://raw.githubusercontent.com/deadhoes/nocons/main/icons/iconname.lua"))()

Available Icons

  • Heart

    loadstring(game:HttpGet("https://raw.githubusercontent.com/deadhoes/nocons/main/icons/heart.lua"))()

  • Fire

    loadstring(game:HttpGet("https://raw.githubusercontent.com/deadhoes/nocons/main/icons/fire.lua"))()

Loader Script (Optional)

Instead of calling individual icons directly, you can load this loader.lua script and use it to load any icon by name: 

local loadIcon = loadstring(game:HttpGet("https://raw.githubusercontent.com/deadhoes/nocons/main/loader.lua"))()
loadIcon("heart")
