Installation

Get started with Nocons

Choose your preferred method to add Nocons to your project.

NPM
npm install nocons

After installation, you can import individual icons:

import { Home, Alert } from 'nocons';
CDN
<script src="https://unpkg.com/nocons"></script>

After adding the script tag, icons will be available under the Nocons global variable.

SVG Download

Download the complete set of SVG icons to use directly in your project.
